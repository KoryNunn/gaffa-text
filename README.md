# gaffa-text

text view for gaffa

## Install:

    npm i gaffa-text

## Add to gaffa:

    gaffa.registerConstructor(require('gaffa-text'));

# API

## Properties (instanceof Gaffa.Property)

### text (get set)

The views text.

### visible (get set)

The views visibility. Actually just renders an empty string in place of `text`
